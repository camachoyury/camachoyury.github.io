Los que desarrollamos apliaciones de escritorio o web estamos acostumbrados al momento de trabajar con bases de datos  tener a mano nuestro administrados de base de datos para poder navegarn entre los datos y la estructura de nuestra base de datos, pero esto cambia cuando trabajas a nivel de dispositivos moviles en este caso Android.
El problema viene dado porque no tenemos un administrador estandar donde podemos navegar en la base de datos SQLite, pero exiten muchas herramientas que nos ayudan, no en gran magnitud, pero sirve en este caso les hablo de un plugin para eclipse que es muy sencillo de instalar: Questoid SQLite Browser.

los pasos para instalar son:
1 Descargar el plugin  sqlitexmlbrowser_1.2.0.zip

2 Descomprimir el .zip

3 Copiar el .jar en el directorio de plugins de eclipse (e.g. /usr/lib/eclipse/plugins)

4 Reiniciar Eclipse.

5 Levantar el emulador android

6 habilitar la perspectiva DDMS >>>> en eclipse ir a Window > Open Perspective > DDMS

7 En esta perspectiva ir a file explorer y buscar nuestra base de datos que usualmente es un .db (e.g. 'data -> data -> com.myproject -> databases -> myproject)
