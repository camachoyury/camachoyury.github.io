ButterKnife es una libreria de inyeccion de vistas para android que ahorra un montón de lineas de código, y es de un fácil uso y configuración. en este post se mostrara como agregar a un proyecto y como usarlo. hace unos días atras Jakewarton creador de esta librería anunciada la salida de la nueva version que usare como ejemplo

para agregar ButterKnife a un proyecto primero de debe agregar algunas lineas de configuración a Gradle
se necesita agregar a build.gradle una dependencia.

dependencies {
classpath 'com.neenbedankt.gradle.plugins:android-apt:1.8'
}
Luego se debe agregar en app/build.gradle

apply plugin: 'android-apt'
android {
...
}
dependencies {
compile 'com.jakewharton:butterknife:8.2.1'
apt 'com.jakewharton:butterknife-compiler:8.2.1'
}

una vez agragradas y sincronizadas las dependencias empezamos a usar la libreria.

vemamos ambos escenarios, sin y con la libreria

vamos a agregar un FloatingActionButton a una activity

SIN ButterKnife:


public class MainActivity extends AppCompatActivity {

FloatingActionButton fab;

@Override
protected void onCreate(Bundle savedInstanceState) {
super.onCreate(savedInstanceState);
setContentView(R.layout.activity_main);

fab = (FloatingActionButton) findViewById(R.id.fab);
fab.setOnClickListener(new View.OnClickListener() {
@Override
public void onClick(View view) {
Snackbar.make(view, "Replace with your own action", Snackbar.LENGTH_LONG)
.setAction("Action", null).show();
}
});
